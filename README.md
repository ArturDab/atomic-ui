# Atomic UI

Centralna biblioteka komponentow React + Tailwind + Shadcn/ui.

## Stack
- React 18 + TypeScript + Vite 5
- Tailwind CSS v3 + Shadcn/ui (Radix UI)
- React Router DOM v6
- DM Sans + JetBrains Mono

## Uruchomienie

```bash
npm install
npm run dev
```

## Struktura

```
src/
├── components/ui/   # Komponenty bazowe (Shadcn-compatible)
├── registry/        # Rejestr z metadanymi i przykladami
└── showcase/        # UI przegladarki biblioteki
```
